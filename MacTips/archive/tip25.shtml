Tip #25, Newer than 8!

MacTip #24 discussed the fact that there have been a number of updates since Mac OS 8, how to acquire, and how to install them.

This MacTip simply points you to a new page at The Macintosh Guy's site. This page is designed to be a one stop spot for all the officially released updates from Apple. I find that when I'm installing a new system it is handy to have access to all the updates in one place instead of trying to remember what is where. I hope this will be a valuable resource for you now and in the future.

And now head over to Eight.Six-Plus! to see if there is any updates that have slipped past in the past couple months.
